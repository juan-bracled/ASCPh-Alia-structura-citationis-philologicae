Estructura básica o esquelo de sistema bibliográfico de citado y composición de bibliografías, compuesto íntegramente para LaTeX, pero que puede usarse, siguiendo la estructura, para otros programas como Microsoft Word. Se compuso con la fuente [AltStandard](https://github.com/Ioannes-Brclds/AltStandard.git). Se puede cambiar la estructura como se considere necesario y hacer un "fork" con modificaciones, con la condición de cambiar el nombre del sistema (se puede cambiar la licencia también, siempre y cuando se respete que sea "código abierto"). Si tiene alguna sugerencia, compártala.
